# Civics Files

This directory now separates the readable book from the raw archive and earlier
working materials.

## Primary Files

- `civics_book.md`
  The current readable textbook / handbook version.
- `all_civics_notes_archive_full.md`
  The full preserved raw archive.

## Supporting Files

- `civics_bootstrapping_notes.md`
  Bootstrapping-specific notes that informed the main book.
- `civics_early_outline.txt`
  An early outline / prose draft from before the current structure.
- `civics_legacy_extract.md`
  A short earlier extract of important ideas from the raw notes.

## Archive Material

- `archive_split_notes/`
  The old split-up archive chunks of the full notes.
- `archive_split_notes/simplified/`
  Older simplified chunk outputs derived from the split notes.

## Reading Order

1. `civics_book.md`
2. `civics_bootstrapping_notes.md` if you want more startup context
3. `all_civics_notes_archive_full.md` if you need the full historical source
